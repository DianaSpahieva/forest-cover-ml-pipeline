# 🌲 Forest Cover ML Pipeline

An end-to-end machine learning pipeline for predicting forest cover types from cartographic and environmental features.

## Project Overview

This project demonstrates a complete supervised machine learning workflow, from exploratory data analysis (EDA) to model development and evaluation. The objective is to accurately classify forest cover types using cartographic variables such as elevation, slope, aspect, soil type, and wilderness area.

## Highlights

- Built an end-to-end machine learning pipeline for multi-class classification
- Performed feature engineering and data preprocessing
- Compared multiple classification algorithms
- Applied 5-fold cross-validation for robust model evaluation
- Optimized a Decision Tree model through hyperparameter tuning
- Achieved **78.1% cross-validation accuracy** after tuning model parameters
